## Getting Started

Ensure that you have `bun` installed using:

```bash
curl -fsSL https://bun.sh/install | bash
```

Create .env file

```bash
cp .env.example .env
```

Then, run the development server:

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
